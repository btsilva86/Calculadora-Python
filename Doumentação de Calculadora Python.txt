Este código é uma calculadora básica que permite ao usuário escolher entre quatro operações matemáticas: soma, subtração, multiplicação e divisão. Abaixo, segue uma explicação passo a passo:

Etapas do Programa:
Exibição do Menu de Operações: O programa exibe quatro opções para o usuário escolher:

Soma

Subtração

Multiplicação

Divisão

Entrada da Escolha do Usuário: O usuário deve digitar o número da operação desejada (1, 2, 3 ou 4).

Entrada dos Números: O programa solicita dois números que serão utilizados no cálculo.

Cálculo das Operações: O programa realiza os quatro cálculos (soma, subtração, multiplicação e divisão) com os números fornecidos, mas só exibirá o resultado da operação escolhida pelo usuário.

Decisão com if: Usando estrutura condicional (if, elif, else), o programa verifica qual operação foi escolhida e imprime o resultado correspondente.

Verificação de Opção Inválida: Caso o usuário digite um número diferente de 1 a 4, o programa exibirá "Opção inválida!".
